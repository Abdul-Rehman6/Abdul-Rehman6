### What I do

I work as an AI Engineer at Chainfore in Lahore, where I build GenAI products that run in production, not in notebooks. Two systems I have shipped:

- **Auxee** — an enterprise GenAI chatbot on Azure (Azure OpenAI, Azure AI Search, RAG over private company data)
- **Falconic AI** — an autonomous software engineering platform built on LangGraph agents with sandboxed code execution (E2B)

Most of my public repos here are focused explorations of the same ideas: multi-agent supervision, agentic RAG, MCP clients and servers, and production-grade LangGraph patterns.
